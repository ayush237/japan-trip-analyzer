# HTML Formatter Rules

1. The output must be styled as a clean, responsive HTML table.
2. Design aesthetics: Use a pastel color palette and clean design. Include a dark navy blue header titled "Bonus Offbeat Spots & Alternatives".
3. Columns required: Spot Name, Location, Description, Booking & Access, Distance / Transit, Timing / Hours, Highlights & Romantic Features, Original Reel Link.
4. The "Highlights & Romantic Features" column must be a clean bulleted list emphasizing elements like scenic walks, serene environments, and premium onsen details.
5. Provide a standalone HTML file (`travel_spots.html`) with embedded CSS for styling.
