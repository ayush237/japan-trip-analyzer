# Data Validation Rules

1. Ensure the spot name and location are accurately extracted.
2. The description must focus on autumn foliage, isolated nature spots, and classy/romantic experiences.
3. Access information MUST NOT be "N/A". If the video doesn't explicitly state the public transport station or exit, use web search to find exact details.
4. Booking price MUST NOT be "N/A". Ensure November pricing (in JPY) is found via web search if missing from the video.
5. All fields (Distance/Transit, Timing/Hours) must be populated with accurate information.
